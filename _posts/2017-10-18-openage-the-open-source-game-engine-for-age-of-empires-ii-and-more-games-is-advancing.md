>Tags: Open Source, Game EngineAnother open source game engine has been advancing recently, this time it&#39;s openage [Official Site, GitHub], the game engine for playing games like Age of Empires, Age of Empires II (HD) and Star Wars: Galactic Battlegrounds. Shared on reddit was this video showing off some of the progress: Looks like it&#39;s really starting to come together, which is awesome. As someone who spent hours playing AoE, this project is really exciting to me. Who knows, in another year perhaps this might be quite playable? Looks a bit rough right now of course, but very interesting to see the state of it right now. Much like OpenMW (Morrowind), OpenRA (Red Alert, Command &amp; Conquer, Dune 2000) and others, it aims to be an engine reimplementation that requires the original game assets. According to a comment from a developer, they&#39;re targetting Age of Empires II first, with Age of Empires and Star Wars: Galactic Battlegrounds to follow....
